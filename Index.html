Overview
What this project is, how the data was collected, and what this dashboard shows, and how to navigate it
🎯

What is This Project?

This is a research project conducted at the University of Guelph under Dr. Sadaf Mollaei. The goal is to systematically document what food sustainability programs exist across all Canadian universities, generate counts and patterns from that data, and use those findings to support the claims made in our literature review.

🔍

What is this Environmental Scan?

An environmental scan is a structured review of publicly available information across every institution. For this project, the websites, sustainability pages, dining services pages, news releases, Instagram accounts, and third-party sources of all 97 scanned institutions were searched and coded.

📊

How was the Data Coded?

We first conducted a literature review of the articles listed in the References section below. From that review we extracted key themes, patterns, and gaps related to food sustainability in university settings. Using those findings, we then coded each Canadian university Yes or No across 8 food sustainability program categories to identify how common each program type is, where overlaps exist, and where gaps remain.

⚠️

Important Limitations

This scan reflects only what universities publish publicly. Programs that exist but are not communicated online are not captured. Outcome data (e.g. % waste reduced, % local purchases) was almost never publicly reported. This is itself a key finding. Data reflects institutional websites as of June 2026.

📋 Methodology: How the Scan Was Conducted

Step 1: University List
Identified all 97 institutions using the official Universities Canada member list as the starting point.
Step 2: Website Search
Searched each university website, sustainability pages, dining services pages, news releases, Instagram accounts, and third-party sources.
Step 3: Coding
Coded each university Yes or No across 8 program categories. If information was unclear or absent, it was coded No Program. Source links were recorded as evidence.
Step 4: Analysis
Calculated counts and percentages per category. Conducted overlap analysis to identify which categories appear together. Summarized themes, gaps, and implications.

Key Findings at a Glance

  • 72.2% of all scanned institutions (70 of 97) have at least one identifiable food sustainability program
  • Sustainable Procurement is the most common program type, found at 51.5% of all institutions
  • Plant-Based Eating programs are the rarest despite being high-impact, at only 22.7% of institutions
  • Food Security and Sustainability programs almost never overlap. They tend to operate in separate silos
  • Almost no university publicly reports measurable outcomes (e.g. % waste diverted, % local food purchased)
  • BC's Feed BC and Quebec's Aliments du Québec create the clearest provincial clusters of documented activity
  • Universities with one program tend to grow into multiple areas. 56 institutions have 3 or more program types

Data Visualizations
Three different ways to look at the same data. Each tells a slightly different story.

① Radar Chart: Shape of Food Sustainability Across Canada

Shows the overall "fingerprint" of Canadian university food sustainability. Each axis is one program category. The further out the point, the more common that program type is. A perfectly round shape would mean all categories are equally common. The gaps show where universities are lagging.

② Bubble Chart: Universities by Province and Program Breadth

Each bubble represents a province. The horizontal position shows how many institutions have any program. The vertical position shows the average number of program types per institution. Bubble size reflects the total number of universities scanned in that province. Provinces in the top-right corner have both high adoption and broad program variety.

③ Stacked Bar: Program Types by Province

Shows which program types are present in each province and how they stack up. Each colour is one program category. Taller bars mean more total program coverage in that province. You can see at a glance how BC and Quebec compare to provinces like Saskatchewan or PEI.

Why a Unified Tool is Needed
What this environmental scan reveals about the need for a standardized assessment approach

🔍 Programs exist but cannot be compared

Food sustainability initiatives are happening across Canadian universities, but they are described using inconsistent language, measured differently (or not at all), and communicated through different channels. Without a common framework, it is impossible to know which approaches are most effective or how institutions compare to each other.

📊 Outcome data is almost never reported publicly

Across 97 scanned institutions, very few publish measurable results. Most describe programs without reporting what those programs actually achieved. A unified tool would establish minimum reporting standards so that universities can demonstrate impact rather than simply listing initiatives.

🏛️ Institutional silos prevent integrated approaches

Food security and sustainability offices rarely work together. Certification programs and procurement programs exist at the same institution without connecting to each other. A tool that maps all program categories in one place could help institutions see where integration opportunities exist.

🌱 The data shows readiness for a tool

72.2% of scanned universities already have at least one food sustainability program. The infrastructure exists. What is missing is a consistent way to assess, quantify, and communicate it. A unified assessment tool would allow universities to benchmark their efforts, identify gaps, and set goals grounded in comparable data.
